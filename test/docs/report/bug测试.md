# Bug 测试结果

## Bug #1：答案判别状态异常

### Bug 描述

进行错误答案测试后，再次输入正确答案时，系统无法正确判别。

### 测试截图

![Bug1-错误答案测试](./assets/bug1_wrong_answer.png)
![Bug1-正确答案测试](./assets/bug1_correct_answer.png)

### 原因分析

系统记录错题的时候把错误的答案数据也记录了下来，没有进行释放。再次输入其他答案时，系统检测到的只有错误的数据，没有接收到新输入的答案。

### 解决方案

记录错题时将错误数据释放，仅记录错误题目的题号信息，避免干扰后续答题判断。

### 状态

✅ 已修复

---

## Bug #2：死循环导致系统崩溃

### Bug 描述

进行超时测试时，刻意输入死循环代码会导致系统无法运行。

### 测试截图

![Bug2-死循环测试](./assets/bug2_infinite_loop.png)

### 原因分析

系统检测输入答案判断对错时，会将代码全部执行一遍才能进行判断。当输入死循环代码时，系统无法跳出循环，导致系统卡死或崩溃。

### 解决方案

更换检测方法：
- 系统检测出错误或无意义的循环时直接跳出
- 显示具体错误原因，不执行完整代码

### 状态

✅ 已修复

---
